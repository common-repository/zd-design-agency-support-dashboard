New Beta Plugin Adds a dashboard support system for ZD Design Agency members WP Powered Website Service Admin Dashbaord and widget area (under Active Development). WP develper and management services so they can keep there clients informed and happy :)

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. All Done :) u can access the the widget and Dash Page from the WP admin Dashboard .
